main.css is the main css file of this website. It defines the main styles of the website used across many pages, such as header, footer, etc. A brief description of main.css is described below.:


the Raleway font combo

one of the important characters in a website is font. Raleway is a modern font used in our website. It is available in 3 variants.


/* raleway font combo */
@font-face{/* font import - raleway */
 font-family: 'raleway';
 src: url('/bistro/fonts/raleway.ttf');
}
@font-face{/* font import - raleway light*/
 font-family: 'raleway light';
 src: url('/bistro/fonts/raleway-light.ttf');
}
@font-face{/* font import - raleway-bold */
 font-family: 'raleway bold';
 src: url('/bistro/fonts/raleway-bold.ttf');
}


@font-face is used to import fonts either from the web or from the server itself. We have included the font face from the fonts folder.

src defines where the source file is. Source file is the default extension for fonts i.e. .ttf (TrueTypeFont).


Learn more about fonts here.

whole page and body

This part represents the whole page and body css. '*' represents the whole page. As described above, raleway and alternative font, cursive is used. The margin and padding of the body is set to 0 or simply reset.


*{
 font-family: 'raleway',cursive;!important
}
body{/* main body css */
 margin: 0;
 padding: 0;
}

mega menu css

This part emphasizes on all about the mega menu.


the top level css

All of the top level css of mega menu applying to all the areas of mega menu.


/* mega menu css */
.menu-large {
  position: static !important;
}
.megamenu {
  padding: 20px 20px;
  width: 100%;
}

 

In .menu-large, the position is set to static. It sets the position of menu to be static.

.megamenu adds the padding of 20px on all sides into itself & specifies that the width of the megamenus is 100%.


CSS Combinator: A combinator is something that explains the relationship between the selectors. It is used to select certain CSS elements (classes, ids, tags) etc according to our need.


Child Selector (>): The child selector selects all elements that are the children of a specified element.


Learn more about combinators here.

& this example explains child selector pretty well.

items & links

This section consists of css styling the different sub components of menu such as div, ul, li, etc.


.megamenu > div > li > ul {
  padding: 0;
  margin: 0;
}
.megamenu > div > li > ul > li {
  list-style: none;
}
.megamenu > div > li > ul > li > a {
  display: block;
  padding: 3px 20px;
  clear: both;
  font-weight: normal;
  line-height: 1.428571429;
  color: #333333;
  white-space: normal;
}
.megamenu > div > li > ul > li > a:hover,
.megamenu > div > li > ul > li > a:focus {
  text-decoration: none;
  color: #262626;
  background-color: #f5f5f5;
}

 

.megamenu > div > li > ul is the css for the all of the actual menu item group. They are given margin and padding of 0.

.megamenu > div > li > ul > li is the css for the all of the actual menu items. Their list style is removed.


.megamenu > div > li > ul > li > a is the css for the all of the actual link inside menu item. It is a given display: block; & padding: 3px 20px;.

Display of block is required for the padding. clear: both; is the css for clearing floats.
  font-weight: normal; is for providing normal font weight to the text.
  line-height: 1.428571429; is for proper vertical alignment of the text.
  color: #333333;
  white-space: normal; is used for wrapping of text. See here.


Both,hover & focus are used to select the "a" tag when it is either focused or hovered in some way. text-decoration: none; is used for removing decorations from the text or removal of blue colour & underline to make it look like normal text and get rid of the boring colour.

It provides the disabled look to the link & header is style by .megamenu.dropdown-header.

responsive-ness

When the screen width is less than or equal to 768px:\


@media (max-width: 768px) {
  .megamenu {
    margin-left: 0;
    margin-right: 0;
  }
  .megamenu > li {
    margin-bottom: 30px;
  }
  .megamenu > li:last-child {
    margin-bottom: 0;
  }
  .megamenu.dropdown-header {
    padding: 3px 15px !important;
  }
  .navbar-nav .open .dropdown-menu .dropdown-header {
    color: #fff;
  }
}

 

margin are removed. A margin bottom of 30px is added to all 'li' childs of .megamenu except last child. drop-down-header padding is adjusted and also color.
