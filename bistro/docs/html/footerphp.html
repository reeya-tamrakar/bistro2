html

All of the html of the footer section is wrapped under <footer> tag.


Footer consists of two parts:

a) Main Footer

b) Copyright Part

a)Main Footer

The footer displayed above copyright part and below the whole content is the main footer.


<div class="foot bg-light"><!-- Light Background Footer -->
       <div class="row">
               <div class="col-sm-6 col-md-3 p-3">
                   Box
                 </div>


It is given a class of Bootstrap footer class foot. bg-light gives it a light background.


Footer is a basically a row with 4 equal width columns. It's what <div class="row"> depict to bootstrap. Out of 4 boxes, 1 box is shown above. All of the boxes have a similar property.


<div class="col-sm-6 col-md-3 p-3"> is what adds a responsive property to this section. In screens of medium size i.e 768 px or less upto small size i.e 576 px, each box occupies 6 units of screen space i.e. 2 fit in a row. Similarly, for greater than 7px, it occupies 3 units of screen space i.e. all 4 fit in the row & finally, for screens less than 576px of width, a box occupies the whole row. p-3 adds a padding of 3px around the content.


The heading of the each box is <h5> wrapped. In 2nd, 3rd & 4th boxes, content along with link, form & social icon bar is present.

Each of the items are Bootstrap default elements, which make it more easier to understand as well as implement.


Learn & Compare the codes of the box elements with bootstrap documentation & find out which elements are used & how.


The icons used in the part are friconix.js icons. One of the example is shown below:


<i class="fi-xwsuxl-youtube"></i>


fi-xwsuxl-youtube is the YouTube icon provided by Friconix. We give the classname of the icon to <i> element to obtain that icon.


Browse more icons or learn about icons present in the footer at the official website of friconix.

b)Copyright Part

Copyright is same in all screens, i.e. occupies a full row width in all screens.


<div class="copy bg-dark p-2"><!-- Copyight text -->
 <p class="text-white" align="center">&copy;2019 Bistro All Rights Reserved.</p>
</div>



copy defines that it's class is copyright. bg-dark makes the background black & p-2 adds a padding of 2px around the content.


text-white makes the color of the text white. &copy; is the entity for adding copyright symbol ©.
